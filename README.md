# Movie-Recommendation-System-

Hey This is one of my first projects as i dive into machine learning 
Built a basic Movie Recommendation System that uses cosine angles to recommend related to movies to the one user enters 
Uses StreamLit for FrontEnd (btw yes i have included a timdb API key publicly in the code credits to campusX for the free API key and the project itself)
The program itself needs two files movie_dict.pkl and similarity.pkl which can obtained using the pickle module from the provided python notebook , Yes the notebook is messy will be making changes soon to improve it !!
